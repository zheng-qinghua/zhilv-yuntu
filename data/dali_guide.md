# 大理旅游完全攻略

## 目的地简介

大理位于云南省西部，洱海之滨，苍山脚下。以白族文化、古城风貌和自然风光闻名，是中国最受欢迎的旅游目的地之一。全年气候温和，四季如春，最佳旅行时间为3-5月和9-11月。

## 大理古城

大理古城始建于明洪武十五年（1382年），是国务院公布的首批24个历史文化名城之一。古城东临洱海，西枕苍山，城内保存有完整的古城墙、城楼和棋盘式街道格局。

### 古城必逛

- 人民路：古城最热闹的步行街，汇集了手工艺品店、咖啡馆、小酒馆
- 洋人街：护国路上段，因早年来此旅游的外国人而得名，酒吧和西餐厅集中
- 复兴路：古城主街，连接南门和北门，两旁商铺林立
- 五华楼：古城标志性建筑，登楼远眺苍山洱海全景
- 南门城楼：大理古城正门，气势恢宏，适合拍照

### 古城美食

- 大理酸辣鱼：洱海活鱼烹制，酸辣鲜美
- 乳扇：白族传统小吃，烤乳扇外酥里嫩
- 大理砂锅米线：汤底浓郁，配料丰富
- 喜洲粑粑：外酥内软的传统面点
- 大理凉鸡米线：夏季消暑佳品

## 洱海生态廊道

洱海生态廊道全长129公里，是环洱海流域生态保护的重要工程，也是游客最喜爱的骑行路线。廊道禁止机动车通行，只允许步行和自行车。

### 推荐骑行段

- 才村-龙龛码头段：约8公里，沿途可远眺苍山，近观洱海水鸟，日出时分非常适合拍照
- 龙龛-喜洲段：约15公里，途经多个白族村落，田园风光绝美
- 海东-双廊段：约12公里，悬崖公路视野开阔，日落时分金光洒满洱海

## 双廊古镇

双廊古镇位于洱海东北岸，面朝苍山十九峰，是观赏苍山洱海全景的绝佳位置。曾登上国家地理杂志封面。

### 双廊必去

- 玉几岛：杨丽萍太阳宫所在地，三面环海，建筑风格独特
- 南诏风情岛：需乘船上岛，岛上有南诏文化展示和热带植物园
- 海街：双廊最热闹的沿海步道，傍晚时分散步看日落
- 大建旁村：保留了最原始的白族渔村风貌

## 喜洲古镇

喜洲是大理白族民居保存最完好的古镇，被誉为"白族民居建筑博物馆"。

### 喜洲亮点

- 严家大院：白族民居建筑典范，"三坊一照壁""四合五天井"格局完整
- 喜洲稻田：春秋两季稻田风光绝美，是摄影爱好者的天堂
- 喜洲粑粑老店：甜咸两种口味，是喜洲最出名的美食
- 蝴蝶泉：每年农历四月十五有蝴蝶会

## 苍山

苍山又名点苍山，是横断山脉最南端的主峰，最高海拔4122米。苍山十九峰巍峨壮丽，每两峰之间有一条溪流注入洱海，形成了著名的"苍山十八溪"。

### 苍山玩法

- 洗马潭索道：大理最长的索道，可到达海拔3900米的洗马潭，高山杜鹃季（5-6月）非常壮观
- 感通索道：到达感通寺，可走玉带云游路徒步约4公里，适合轻松徒步
- 中和寺：苍山山腰的古寺，是俯瞰大理坝子和洱海的最佳观景台之一

### 苍山注意事项

- 海拔较高，注意防寒和防晒
- 索道运营时间一般为8:30-17:00
- 高山区域氧气稀薄，老人和儿童需谨慎

## 崇圣寺三塔

崇圣寺三塔是大理的标志性景观，始建于南诏时期（约9世纪），已有1200多年历史。主塔千寻塔高69.13米，是中国现存最高的古塔之一。三塔倒影在寺前的聚影池中，是大理最具代表性的画面。

门票：75元/人
开放时间：8:00-18:00

## 行程参考

### 3天经典行程

**第1天**：大理古城 → 崇圣寺三塔 → 人民路夜市
**第2天**：洱海生态廊道骑行（才村-喜洲段）→ 喜洲古镇 → 双廊看日落
**第3天**：苍山（感通索道+玉带路徒步）→ 大理大学樱花（3月限定）

### 5天慢节奏行程

**第1天**：抵达大理 → 大理古城闲逛 → 五华楼看古城全景 → 人民路夜市
**第2天**：洱海生态廊道骑行（才村-喜洲全程）→ 喜洲古镇 → 喜洲稻田拍照 → 住喜洲
**第3天**：喜洲 → 双廊古镇 → 玉几岛 → 南诏风情岛 → 海街日落
**第4天**：苍山洗马潭索道 → 高山杜鹃 → 中和寺 → 寂照庵吃素斋
**第5天**：崇圣寺三塔 → 大理大学 → 购买伴手礼 → 返程

## 交通信息

- 大理机场距古城约30公里，打车约40分钟，费用约60元
- 大理火车站距古城约15公里，打车约20分钟，费用约30元
- 古城内步行即可，去洱海边可租电动车（约50元/天）或自行车（约20元/天）

## 住宿推荐

- 古城内民宿：价格150-500元/晚，方便夜逛古城
- 洱海边客栈：价格300-800元/晚，推窗见海
- 双廊海景房：价格500-1500元/晚，适合预算充裕的旅行者
- 喜洲田园民宿：价格200-400元/晚，安静舒适

## 最佳拍照点

- 龙龛码头日出：清晨6:30-7:30，水面平静如镜
- 双廊海街日落：傍晚18:30-19:30，苍山金色时刻
- 喜洲稻田：8-9月金黄一片，4-5月绿油油
- 崇圣寺三塔倒影：上午9:00-11:00光线最佳
