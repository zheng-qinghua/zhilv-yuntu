# 智旅云图 — 启动指南

## 环境要求

- Python 3.11+
- MySQL 8.0+（已安装并启动）
- Windows 10/11

## 数据库配置

MySQL 连接信息（`.env` 文件）：

```
MYSQL_HOST = localhost
MYSQL_PORT = 3306
MYSQL_DATABASE = travel_agent
MYSQL_USER = root
MYSQL_PASSWORD = 123456
```

## 首次启动步骤

### 1. 安装依赖

```bash
cd "C:\Users\20243\Desktop\Agent Param"
pip install -r requirements.txt
```

### 2. 导入攻略数据（首次必须执行）

将 `data/` 目录下的攻略文档切分、向量化、存入 MySQL 并构建 FAISS 索引：

```bash
python main.py --ingest
```

执行后会输出类似：

```
[ingest] 共切分 68 个文本块
[ingest] 正在嵌入 68 条文本...
[ingest] 已写入 MySQL 68 条记录
[ingest] FAISS 索引构建完成，共 68 个向量
[ingest] 全部完成！
```

> 如果数据库已有数据，可跳过此步骤。

### 3. 启动 Web 服务

```bash
uvicorn app:app --host 127.0.0.1 --port 8000
```

或者让局域网内其他设备也能访问：

```bash
uvicorn app:app --host 0.0.0.0 --port 8000
```

### 4. 打开浏览器

访问: **http://localhost:8000**

## 使用示例

登录后，在聊天框输入以下内容：

### Q&A 模式
```
大理有什么值得去的景点？
推荐几个西安必吃的美食
三亚几月份去最好？
```

### 行程规划模式
```
帮我规划一个成都3天行程，预算3000
6月10号去大理，两个人，预算5000，喜欢古镇和拍照
```

助手会识别意图，行程规划模式会调用完整的规划引擎生成日程、预算和交通建议；Q&A 模式会基于本地攻略知识库回答。

每条助手回复末尾可点击"查看参考来源"展开 RAG 检索到的攻略原文。

## 常见问题

### 端口被占用

换一个端口启动：

```bash
uvicorn app:app --host 127.0.0.1 --port 8001
```

### 数据库连不上

确认 MySQL 服务已启动：

```bash
# Windows
net start MySQL80
```

### FAISS 索引不存在

首次使用前必须运行 `python main.py --ingest`。

### 重新导入数据

```bash
python -c "from database.connection import engine, Base; from database.init_db import *; Base.metadata.drop_all(bind=engine); Base.metadata.create_all(bind=engine); print('done')"
python main.py --ingest
```
